### 1. 底部TabBar华丽 redesign

#### 视觉升级
- **浮动设计**: TabBar改为浮动胶囊式设计，底部居中，不再占满全宽
- **玻璃态效果**: 强化毛玻璃效果，带渐变边框
- **激活状态**: 
  - 选中项放大并上浮
  - 彩色渐变背景 + 发光效果
  - 图标动画（弹跳/旋转）
  - 文字显示/隐藏动画
- **未选中状态**: 简洁图标，hover时微光效果

### 2. 多语言支持 (i18n)

#### 技术方案
- 使用 `react-i18next` 库
- 语言文件结构:
  ```
  src/i18n/
  ├── index.ts          # i18n配置
  ├── locales/
  │   ├── zh-CN.json    # 简体中文
  │   ├── en-US.json    # 英文
  │   └── ja-JP.json    # 日文（预留）
  ```

#### 实现内容
- **语言切换器**: Header右上角添加语言切换下拉菜单
- **自动检测**: 根据浏览器语言自动设置
- **持久化**: 使用 localStorage 保存用户选择
- **全站翻译**: 
  - 所有页面标题和内容
  - 所有按钮和标签
  - 所有提示和错误信息
  - 所有弹窗内容

#### 翻译键值设计
```json
{
  "nav": {
```
